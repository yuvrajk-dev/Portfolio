A modern developer portfolio built with **React.js**, **Tailwind CSS**, and **Motion.dev** to showcase my projects, technical skills, and experience. The portfolio also features a fun **real-time cursor sharing** experience powered by **Socket.IO**, allowing connected visitors to see each other's mouse movements live.

🔗 **Live Site:** https://yuvrajk-dev.vercel.app

---

## ✨ Features

- 🎨 Modern and responsive portfolio design
- 🎞️ Smooth page transitions and scroll-triggered animations with Motion.dev
- 🖱️ Real-time cursor sharing using Socket.IO
- ⚡ Node.js + Express backend for WebSocket communication
- 📂 Dedicated sections for projects, skills, and contact information
- 📱 Fully responsive across desktop, tablet, and mobile
- ♿ Semantic HTML and accessible component structure
- 🚀 Frontend deployed on Vercel and backend deployed on Render

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/yuvrajk-dev/portfolio.git
```

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
node index.js
```
